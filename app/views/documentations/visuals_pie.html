Kuchendiagramm

Mit dem Kuchendiagramm können Anteile am Gesamten dargstellt werden.

Eine Abfrage der Form SELECT geschlecht, anzahl FROM todesfaelle; liefert die folgende Darstellung:

Die Abfrage sollte immer genau zwei Spalten liefern; die erste Spalte wird für die Beschriftung verwendet, die zweite um den Anteil zu bestimmen. Abfragen, die mehr als zwei Spalten liefern, funktionieren nicht!

Das Kuchendiagramm kann gestylt werden, indem du die verschiedenen Abschnitte ansprichst. Der erste Abschnitt hat immer die Klasse .ct-series-a, der zweite .ct-series-b und so weiter. Die Abschnitte können dann über die Klasse .ct-slice-pie angesteuert werden. Du siehst die Klassen auch im Quellcode der Website wenn du die Abschnitte mit dem Inspektor auswählst. Du kannst die folgenden Eigenschaften ändern:

  • `fill`, die Füllfarbe, siehe Beispiele
  • `stroke`, die Farbe des Rahmens, siehe Beispiele
  • `stroke-width`, die Balkenbreite, siehe Beispiele
  • `stroke-dasharray`, der Linienstil, siehe Beispiele
  • `stroke-linecap`, die Linienenden, siehe Beispiele

Beispiel

    .ct-series-a .ct-slice-pie {
      fill: blue;
      stroke: white;
      stroke-width: 4px;
    }
