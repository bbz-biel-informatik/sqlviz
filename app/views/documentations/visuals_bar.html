Balkendiagramm

Mit dem Balkendiagramm können zeitliche Verläufe dargstellt werden.

Eine Abfrage der Form SELECT datum, temperatur FROM temperaturen; liefert die folgende Darstellung:

Die Abfrage sollte immer zwei oder mehr Spalten liefern; die erste Spalte wird für die Beschriftung verwendet, alle weiteren werden als Balken gezeichnet.

Das Balkendiagramm kann gestylt werden, indem du die verschiedenen Balken ansprichst. Der erste Balken hat immer die Klasse .ct-series-a, der zweite .ct-series-b und so weiter. Die Balken können dann über die Klasse .ct-bar angesteuert werden. Du siehst die Klassen auch im Quellcode der Website wenn du die Balken mit dem Inspektor auswählst. Du kannst die folgenden Eigenschaften ändern:

  • `stroke`, die Farbe, siehe Beispiele
  • `stroke-width`, die Balkenbreite, siehe Beispiele
  • `stroke-dasharray`, der Linienstil, siehe Beispiele
  • `stroke-linecap`, die Linienenden, siehe Beispiele

Beispiel

    .ct-series-a .ct-bar {
      stroke: red;
      stroke-width: 20px;
      stroke-linecap: round;
    }
