Liniendiagramm

Mit dem Liniendiagramm können kontinuierliche zeitliche Verläufe dargstellt werden.

Eine Abfrage der Form SELECT datum, temperatur FROM temperaturen; liefert die folgende Darstellung:

Die Abfrage sollte immer zwei oder mehr Spalten liefern; die erste Spalte wird für die Beschriftung verwendet, alle weiteren werden als Linien gezeichnet.

Das Liniendiagramm kann gestylt werden, indem du die verschiedenen Linien ansprichst. Die erste Linie hat immer die Klasse .ct-series-a, die zweite .ct-series-b und so weiter. Die Linien können dann über die Klasse .ct-line angesteuert werden. Die Datenpunkte über die Klasse .ct-point. Du siehst die Klassen auch im Quellcode der Website wenn du die Linien und Punkte mit dem Inspektor auswählst. Du kannst die folgenden Eigenschaften der Linien und Punkte ändern:

  • `stroke`, die Farbe, siehe Beispiele
  • `stroke-width`, die Balkenbreite, siehe Beispiele
  • `stroke-dasharray`, der Linienstil, siehe Beispiele
  • `stroke-linecap`, die Linienenden, siehe Beispiele

Beispiel

    .ct-series-a .ct-line {
      stroke: red;
      stroke-width: 20px;
      stroke-linecap: round;
    }

    .ct-series-a .ct-point {
      stroke: red;
      stroke-width: 20px;
      stroke-linecap: square;
    }
